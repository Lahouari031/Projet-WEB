Projet Web TI402 – Site vitrine du Département Informatique EFREI – 2025/2026

Projet TI402

Notre projet

Ce site vitrine a été réalisé dans le cadre du cours de Programmation Web TI402 à l'EFREI Paris, sous la direction de M. Mohamed HAMIDI.

Informations projet

  • ▶ Module : TI402 – Programmation Web
  • ▶ Enseignant : M. Mohamed HAMIDI
  • ▶ Année universitaire : 2025/2026
  • ▶ Technologies : HTML5, CSS3, JavaScript

Objectifs pédagogiques

  • ▶ HTML5 sémantique et valide W3C
  • ▶ CSS3 responsive (media queries)
  • ▶ JavaScript interactif et dynamique
  • ▶ Design et maquette (wireframe)

Technologies

Stack technique

🟠

HTML5

Structure sémantique, accessibilité ARIA, formulaires avancés. Validation W3C sur toutes les pages.

🔵

CSS3

Variables CSS, Flexbox, Grid, animations, transitions, media queries pour un design fully responsive.

🟡

JavaScript

Vanilla JS : Intersection Observer, gestion des événements, animation des compteurs, accordéon FAQ, carrousel, validation formulaire.

JavaScript

Fonctionnalités dynamiques

🎠

Carrousel automatique

Diaporama d'images avec navigation manuelle (boutons précédent/suivant), indicateurs dots et défilement automatique toutes les 5 secondes.

📊

Compteurs animés

Animation des chiffres clés déclenchée à l'entrée dans le viewport via Intersection Observer, pour un effet visuel percutant.

🎭

Scroll Reveal

Apparition progressive des éléments au scroll grâce à Intersection Observer et des animations CSS en cascade.

❓

FAQ Accordéon

Questions/réponses interactives avec ouverture/fermeture animée, une seule question ouverte à la fois.

📋

Validation Formulaire

Validation côté client des champs obligatoires et du format email, avec retour visuel et confirmation de succès.

🍔

Menu Burger

Navigation mobile responsive avec menu hamburger animé et fermeture automatique au clic sur un lien.
